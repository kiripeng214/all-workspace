# PRD: 详情页点击生日快速修改

## 1. 背景

- 当前修改生日需进编辑页，流程长
- 生日数据是日期格式，需要日期选择器而非普通输入框

## 2. 需求

### 2.1 功能

| # | 功能 | 优先级 |
|---|------|--------|
| 1 | 详情页点击生日标签🎂，弹出日期选择弹窗 | P0 |
| 2 | 弹窗包含日期 picker，当前生日预选 | P0 |
| 3 | 确认后保存，页面刷新 | P0 |

### 2.2 流程

```
详情页 → 点🎂生日标签 → 弹窗（日期 picker 预选当前值）
   → 选择日期 → 确认 → API 保存 → 弹窗关 → 刷新
   → 取消 → 弹窗关
```

## 3. 验收标准（含视觉）

- [ ] 点击🎂标签弹出编辑框，弹窗使用与改名弹窗相同的 overlay 方式
- [ ] 弹窗居中，宽度 560rpx（与改名弹窗一致）
- [ ] 日期 picker 在弹窗内，显示当前已选日期
- [ ] 确认按钮绿色 #4CAF50，取消灰色 #f5f5f5
- [ ] 保存后标签立即显示新日期
- [ ] 取消关闭，不修改
