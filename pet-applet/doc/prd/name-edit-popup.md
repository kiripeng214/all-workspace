# PRD: 详情页点击姓名弹出编辑框

## 1. 背景

修改宠物姓名需要跳转完整编辑页，流程太长。期望在详情页直接点击姓名弹窗修改。

## 2. 需求

### 2.1 功能

| # | 功能 | 优先级 |
|---|------|--------|
| 1 | 详情页点击宠物姓名，弹出编辑框 | P0 |
| 2 | 编辑框内修改后提交保存 | P0 |
| 3 | 保存后刷新页面显示新姓名 | P0 |

### 2.2 流程

```
详情页 → 点姓名 → 弹出编辑框（预填当前姓名）
   → 改后点确认 → API 保存 → 弹窗关 → 刷新
   → 取消/点蒙层 → 弹窗关
```

## 3. 验收标准

- [ ] 点击姓名弹出编辑框
- [ ] 弹窗预填当前姓名
- [ ] 改为空时提示，不保存
- [ ] 保存后页面刷新
- [ ] 取消/点蒙层可关闭
